Batch feature gives you the ability to grab episodes' download links, in less time than doing it manually.
The extension programatically sends requests to the episodes and grab the links, then it displays the links, as well as a command line for IDMan for more flexiblity and less work

  1. How to use

    2. After setting your prefered servers order and choosing episodes you want, you have two options:

    1. IDMan command line

      2. This one is recommended. The greatest thing about commands is that you can rename file names in IDMan, in fact, IDMan gets the file name sent by the server and uses it, or if on browser, IDMan grabs the link and uses the page title as the filename


      So the command line tells IDMan what filename should the link have instead of having names like these :

      or most importantly avoid names like these (Rapidvideo)

      in case they came from external websites.
