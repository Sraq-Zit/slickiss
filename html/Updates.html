User-agent for BetaX
  • What is User-Agent ?
    User-agent is a piece of information that serves as an identification for your browser.
    Changing the user-agent might create some kind of compatibility, say, for example you want to download an application for windows and you user-agent was changed to MacOS, the website might link you to the MacOS version by default. However, changing the user-agent is very safe.
    User-agent are usually spoofed to test various environments.
  • Why tho ?
    No idea.
    BetaX uses googlevideo domain to get the videos. Apparently googlevideo returns a 403 (Forbidden) status code when the user-agent is not set, but I don't know why is this restriction.
  • So what now ?
    Either you follow kissanime instructions to change user-agent, or you can let Slickiss handle it.
    Slickiss will change user-agent for Kissanime and googlevideo URLs.
    Enable it from settings: Or you will find some additional instruction in BetaX instructions
